Merge ACF tabs from different goups.

== Description ==

ACF Merge Group Tabs is plugin for Wordpress with installed Advanced Custom Fields. The plugin merges tabs from different groups to one box.

== Installation ==

Installing plugin can be done either by searching for "ACF Merge Group Tabs" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
